Frequently Asked Questions

How do I add Stellar Bot to my server?

▼

Click the "Add to Discord" button on our homepage, then select your server and authorize the bot. You'll need "Manage Server" permissions to add bots to a server.

What permissions does the bot need?

▼

Stellar Bot needs the following permissions to function properly:

  • Send Messages
  • Connect to Voice Channels
  • Speak in Voice Channels
  • Embed Links
  • Additional permissions for moderation commands

How do I set up the DJ role?

▼

Use the !dj command to manage DJ roles:

!dj set @role - Set a role as DJ
!dj remove - Remove DJ role requirement
!dj info - View current DJ settings

What music sources are supported?

▼

Stellar Bot supports playback from:

  • YouTube
  • Spotify
  • SoundCloud
  • Direct URLs

How can I report issues or suggest features?

▼

You can:

  • Join our Support Server
  • Create an issue on our GitHub repository
  • Contact us via email at support@stellarbot.com
