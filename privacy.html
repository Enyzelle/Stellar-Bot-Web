Privacy Policy

Last updated: December 2024

Information We Collect

Stellar Bot collects and stores the following information:

  • Server IDs for configuration settings
  • Channel IDs for music and moderation features
  • User IDs for command cooldowns and permissions
  • Command usage statistics for improvement purposes

How We Use Your Information

The collected information is used to:

  • Provide and maintain bot functionality
  • Save server-specific settings and preferences
  • Improve bot features and user experience
  • Monitor and prevent abuse

Data Storage and Security

All data is securely stored and encrypted. We implement appropriate security measures to protect against unauthorized access, alteration, disclosure, or destruction of your information.

Data Deletion

You can request deletion of your data by:

  • Contacting us through our support server
  • Removing the bot from your server

Third-Party Services

Stellar Bot uses the following third-party services:

  • DiscordJS API
  • Riffy API

Contact Us

If you have any questions about this Privacy Policy, please contact us:

  • Join our Discord Support Server
  • Email us at: support@stellarbot.com
  • Create an issue on our GitHub repository
